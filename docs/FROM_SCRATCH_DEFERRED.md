# From-Scratch Setup Deferred

V1 assumes the user already has OpenClaw installed.

The future from-scratch path should:

- install OpenClaw;
- install LanceDB Pro;
- create the OpenClaw home;
- configure Slack apps;
- render this harness;
- run the same validation suite.

This is intentionally not implemented in v1 so the primary migration path stays tight and safe.
